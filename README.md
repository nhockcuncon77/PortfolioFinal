## Getting started

### Prerequisites

- Node.js 18+
- npm (or yarn / pnpm)

### Install and run

```bash
# Clone the repo
git clone https://github.com/nhockcuncon77/PortfolioFinal.git
cd PortfolioFinal

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Other scripts

| Command        | Description              |
|----------------|--------------------------|
| `npm run build` | Production build        |
| `npm run preview` | Preview production build |
| `npm run lint`   | Run ESLint              |
